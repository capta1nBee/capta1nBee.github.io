eBPF Network Flow Intelligence

eBPF Ağ Akış İstihbaratı

Poyraz-K8s uses high-performance eBPF technology to map complex cluster communication in real-time, operating entirely within the Linux kernel to bypass Kube-proxy bottlenecks.

Poyraz-K8s, Kube-proxy darboğazlarını aşmak için tamamen Linux çekirdeği (kernel) içinde çalışarak karmaşık küme iletişimlerini gerçek zamanlı olarak haritalamak için yüksek performanslı eBPF teknolojisi kullanır.

"Ping-Pong" Zero-Downtime Router

"Ping-Pong" Sıfır-Downtime Yönlendiricisi

Uses a Dual-Table Router (Table 1 / Table 2). While the backend aggregates the analytics from Table 1, eBPF continues writing to Table 2 ensuring packet flow analysis is never blocked.

Çift Tablolu Yönlendirici (Table 1 / Table 2) kullanır. Backend (arka uç) Tablo 1'den analitiği toplarken; eBPF, paket akış analizinin asla engellenmemesini sağlamak için Tablo 2'ye yazmaya devam eder.

Firewall Spec Generator

Güvenlik Duvarı Kuralı Oluşturucu

Analyzes historical flow dimensions (L4/L7) to automatically suggest declarative Kubernetes NetworkPolicy rules to clamp down lateral movement.

Yanal (lateral) hareketleri kısıtlamak amacıyla (declerative) bildirimsel Kubernetes NetworkPolicy kurallarını otomatik olarak önermek için geçmiş akış boyutlarını (L4/L7) analiz eder.
