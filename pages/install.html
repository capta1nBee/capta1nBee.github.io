Platform Installation

Platform Kurulumu

Poyraz-K8s is not a monolithic script; it is a distributed intelligent layer across your Kubernetes clusters. The installation process is divided into two distinct scopes: The Central Intelligence Hub (Backend) and the Edge Agents (eBPF & Security) deployed on each managed cluster.

Poyraz-K8s monolitik bir betik değildir; Kubernetes kümeleriniz (cluster) boyunca dağıtılmış akıllı bir katmandır. Kurulum süreci iki belirgin kapsama (scope) ayrılmıştır: Merkezi İstihbarat Merkezi (Backend) ve yönetilen her kümeye dağıtılan Uç Ajanları (eBPF ve Güvenlik).

1. Central Hub (Backend orchestrator)

1. Merkezi Sistem (Backend orkestratörü)

The core backend provides the API, Casbin Authorization, PostgreSQL persistence, and Helm Bridge. It is installed centrally only once.

Temel backend sistemi; API, Casbin Yetkilendirme, PostgreSQL veritabanı kalıcılığı ve Helm Köprüsünü sağlar. Sadece bir kez merkezi olarak kurulur.

helm install poyraz-backend ./charts

2. Edge Agents (DaemonSets)

2. Uç Ajanları (DaemonSet'ler)

eBPF network scanners and Runtime Security agents are deployed to every cluster you wish to manage, running as priviledged DaemonSets on every node.

eBPF ağ tarayıcıları ve Runtime (Çalışma Zamanı) Güvenlik ajanları, yönetmek istediğiniz her kümeye dağıtılır ve her node (düğüm) üzerinde ayrıcalıklı DaemonSet'ler olarak çalışır.

kubectl apply -f poyraz-agents.yaml
