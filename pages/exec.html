Terminal Exec Guard

Terminal Exec Koruması

Zero-trust execution environment. Every terminal shell requested by a user passes through the Great Firewall—a strict RegEx parser that blocks malicious injections by default.

Sıfır-güven (Zero-trust) çalıştırma ortamı. Bir kullanıcı tarafından istenen her terminal kabuğu (shell), kötü niyetli enjeksiyonları varsayılan olarak engelleyen katı bir RegEx ayrıştırıcısı olan Büyük Güvenlik Duvarı'ndan (Great Firewall) geçer.

The Great Firewall

Büyük Güvenlik Duvarı

Uses defensive normalization to split pipes (|) and explicitly blocks shell operators like ; & > < ` $ || before execution ever reaches the Kubernetes API.

Boru (pipe |) karakterlerini ayırmak için savunmacı bir normalizasyon kullanır ve K8s API'ye ulaşmadan önce ; & > < ` $ || gibi kabuk operatörlerini açıkça engeller.

Casbin Policy Linking

Casbin Politika Bağlantısı

Roles define strictly which regex patterns are allowed. If the parsed base command does not match an allowed pattern in the database, it rejects.

Roller, hangi regex pattern'lerinin izin verildiğini kesin olarak tanımlar. Ayrıştırılan temel komut veritabanındaki izin verilen pattern listesi ile eşleşmezse, (istek) reddedilir.
